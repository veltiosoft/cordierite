# InkscapeでPNGロゴをSVG化して整える手順

## 前提

- Inkscape 1.4.x
- 入力ファイル：PNG
- 出力ファイル：Plain SVG（Web向け）

---

# 1. PNGを読み込む

```
ファイル → インポート
```

または

```
ファイル → 開く
```

PNGを選択する。

> 読み込み時の設定は基本的にデフォルトで問題ない。

---

# 2. PNGを選択する

画像をクリックして選択する。

選択されると画像の周囲にハンドルが表示される。

---

# 3. ビットマップをトレースする

```
パス → ビットマップをトレース
```

### 単色ロゴ

```
単一スキャン
↓
明るさのしきい値
```

しきい値

```
0.45〜0.70
```

程度から調整する。

プレビューを見ながら、

- 線が潰れない
- 欠けない

ように調整する。

### カラーロゴ

```
複数スキャン
```

を使用する。

---

# 4. SVGを生成する

```
適用
```

を押す。

生成されたSVGは元PNGの真上に重なって配置される。

---

# 5. 元PNGを削除する

SVGを少し横へドラッグする。

```
PNG
SVG
```

の2つが存在していることを確認する。

元PNGのみ削除する。

```
Delete
```

---

# 6. パスを分解する

ロゴ全体を選択して

```
パス → 分解
```

ショートカット

```
Ctrl + Shift + K
```

これで各パーツを個別に編集できる。

---

# 7. 不要なパーツを削除する

自動トレースでは

- 小さな点
- ノイズ
- ゴミ

が生成されることがある。

不要なパーツを選択して

```
Delete
```

で削除する。

---

# 8. パスを簡略化する

```
パス → 簡略化
```

ショートカット

```
Ctrl + L
```

アンカーポイントを減らす。

※やりすぎると形が崩れるため1回ずつ確認する。

---

# 9. ノードを調整する

ノードツール

```
N
```

または

```
ノード編集ツール
```

を使用する。

確認ポイント

- 左右対称になっているか
- 曲線が滑らかか
- 不自然な角がないか

不要なノードは削除して整える。

---

# 10. 色を調整する

```
オブジェクト
↓
フィルとストローク
```

設定例

```
フィル：ブランドカラー
ストローク：なし
```

Webロゴでは

```
塗りのみ
```

がおすすめ。

---

# 11. 線幅を調整する（必要な場合）

線を使用しているロゴなら

```
フィルとストローク
↓
ストロークスタイル
```

で線幅を調整する。

最終的には

```
パス
↓
ストロークをパスに変換
```

しておくと表示が安定する。

---

# 12. ページサイズをロゴに合わせる

```
ファイル
↓
ドキュメントのプロパティ
```

または

```
Shift + Ctrl + D
```

```
ページサイズ
↓
ページを描画または選択範囲に合わせる
```

を実行する。

必要であれば数px程度の余白を追加する。

---

# 13. 保存する

```
ファイル
↓
名前を付けて保存
```

保存形式

```
Plain SVG
```

を選択する。

Webサイトでは通常のInkscape SVGよりPlain SVGが推奨される。

---

# Webサイトで利用する場合のポイント

- SVGを使用する
- PNGは配布用・SNS用のみ
- ロゴは単色でも成立するデザインにする
- CSSで色を変更できるようにする

例

```html
<svg class="logo">
    ...
</svg>
```

```css
.logo {
    color: #2563eb;
}

.logo path {
    fill: currentColor;
}
```

これにより

- ライトモード
- ダークモード
- ホバー時

などで色を自由に変更できる。

---

# おすすめワークフロー

```

        ↓

        ↓

        ↓
元PNG削除
        ↓
パス分解
        ↓
不要パーツ削除
        ↓
簡略化
        ↓
ノード修正
        ↓
色調整
        ↓
ページサイズ調整
        ↓
Plain SVG保存
```
